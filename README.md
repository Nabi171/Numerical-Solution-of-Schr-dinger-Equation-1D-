# Numerical Solution of Schrödinger Equation (1D)

## Description
This project simulates the time evolution of a quantum wave packet using the time-dependent Schrödinger equation.

## Methods
- Finite Difference Method
- Crank-Nicolson approximation
- Gaussian wave packet initialization

## Features
- Wave function evolution
- Probability density visualization
- Free particle simulation

## Tools Used
- MATLAB

## Result
The simulation shows spreading of a Gaussian wave packet over time.
